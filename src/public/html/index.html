API Documentation

Complete reference for integrating our captcha API

POST /captcha/generate

Generate a new captcha image

Parameters:

Name Type Required Description
difficulty integer No Difficulty level (1-5)
format string No Output format (png, jpeg, svg)
length integer No Character count (3-8)
POST /captcha/verify

Verify a captcha solution

Parameters:

Name Type Required Description
captcha_id string Yes Captcha identifier
solution string Yes User provided solution
GET /captcha/{id}

Retrieve captcha image by ID

Parameters:

Name Type Required Description
id string Yes Captcha identifier

cURL Example

# Generate captcha
curl -X POST https://api.example.com/captcha/generate \
  -H "Content-Type: application/json" \
  -d '{
    "difficulty": 3,
    "format": "png",
    "length": 5
  }'

# Verify captcha
curl -X POST https://api.example.com/captcha/verify \
  -H "Content-Type: application/json" \
  -d '{
    "captcha_id": "cap_abc123",
    "solution": "ABC123"
  }'

Python Example

import requests

# Generate captcha
response = requests.post('https://api.example.com/captcha/generate', 
    json={
        'difficulty': 3,
        'format': 'png',
        'length': 5
    }
)

captcha = response.json()
print(captcha)

# Verify captcha
verify_response = requests.post('https://api.example.com/captcha/verify',
    json={
        'captcha_id': captcha['id'],
        'solution': user_input
    }
)

Rust Example

use reqwest;
use serde_json::json;

#[tokio::main]
async fn main() -> Result<(), Box> {
    let client = reqwest::Client::new();
    
    // Generate captcha
    let response = client
        .post("https://api.example.com/captcha/generate")
        .json(&json!({
            "difficulty": 3,
            "format": "png",
            "length": 5
        }))
        .send()
        .await?;
    
    let captcha: serde_json::Value = response.json().await?;
    println!("{}", captcha);
    
    Ok(())
}
